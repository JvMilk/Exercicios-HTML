Requisitos

Nível 1: Cartão pessoal

  • Apresentação
  • Interesses profissionais
  • Experiências profissionais
  • Historico acadêmico

Nível 2: Lista de compras

  • Crie uma lista organizada por tópicos (categorias).
  • Utilize caixas de seleção (checkbox) para cada item da lista.
  • Coloque os itens dentro de uma borda ou caixa (container) para melhorar a organização visual.

Nível 3: Galeria

  • Crie uma galeria organizada por tópicos (categorias).
  • Selecione 3 imagens por tópico.
  • Organize as imagens lado a lado.

Nível 4: Formulário

  • Crie um formulário contendo os seguintes campos: Nome, Idade, Nome de Usuário (Username), Senha (Password) e E-mail, além de outros campos adicionais de sua escolha.
  • Cada campo deve ser organizado em uma linha separada, de forma clara e estruturada.
  • No campo E-mail, utilize um placeholder para indicar ao usuário o formato esperado de preenchimento.
