Nível 1: O Básico (Texto e Estrutura)

Desafio 1: O Cartão de Visita Digital

Objetivo: Criar uma página simples sobre você.

Requisitos:

  1. Usar h1 para o seu nome.
  2. Usar h2 para o seu cargo ou hobby principal.
  3. Usar p para escrever um parágrafo sobre você.
  4. Usar strong (ou b) para destacar uma habilidade.
  5. Usar em (ou i) para destacar uma frase engraçada.

Bônus: Adicionar uma linha horizontal (hr) separando o título do texto.
